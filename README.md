# Client Website Management System (WordPress)

## 📌 Overview
This project is a custom WordPress-based CMS built to simulate a real-world client website environment. It focuses on dynamic content management, performance optimisation, and ongoing maintenance workflows.

## 🚀 Features
- Custom WordPress theme built with PHP, HTML, CSS, JavaScript
- Dynamic content using Advanced Custom Fields (ACF)
- Responsive UI built from design concepts (Figma-based)
- Services page with repeater fields
- Performance optimisation (caching, efficient queries)
- Debugging and troubleshooting in a live-like environment
- Maintenance workflows (updates, backups, monitoring)

## 🛠 Tech Stack
- WordPress (Custom Theme Development)
- PHP
- JavaScript (ES6)
- HTML5 & CSS3
- ACF (Advanced Custom Fields)
- Git (Version Control)
